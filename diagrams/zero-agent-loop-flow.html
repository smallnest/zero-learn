agent.Run — Agent 循环流程

turn 循环：分区工具 → 主动压缩 → provider 流式（重连/停顿重试）→ 工具批处理（权限门+沙箱）→ 完成判定

请求前（每轮）

  • • partitionTools：按权限+loaded 定本轮工具集
  • • 超阈值 MCP 工具折叠到 tool_search
  • • maybeCompact 主动压缩最旧中段
  • • streamWithReconnect 连接前重连 + 停顿重试

工具批处理

  • • 逐个 executeToolCall：过滤/spec 门/预拒绝
  • • 权限门 + Sandbox.Evaluate（策略/授权/前缀）
  • • ask_user / request_permissions 循环内拦截
  • • 重复失败守卫 · stopReason 提前结束
  • • 批后一次 SelfCorrect.AfterEdit + 模型升级

完成判定

  • • 交互式：文本即最终答案，立即返回
  • • Headless：自报未完成 / 续接催继续 / 验收检查
  • • 空回合、dropped 调用有专门守卫与 nudge
  • • 超 maxTurns 收尾 · 返回 Result
