zero CLI 运行流程

入口 → 命令分发 → 配置解析 → 两条启动路径 → 共享 Agent 循环与子系统

入口与分发

  • • cmd/zero/main.go → cli.Run
  • • runWithDeps：appDeps 依赖注入 + panic→crash 报告
  • • switch(args[0]) 路由到各 run*
  • • 四类：TUI / exec / 认证配置 / 服务只读

配置解析（六层）

  • • 用户 ~/.config/zero/config.json（macOS 亦然）
  • • 项目 .zero/config.json → 环境变量
  • • provider 命令 → Overrides → normalize
  • • ErrNoActiveProvider 仅 TUI 降级进 setup

Agent 循环与子系统

  • • TUI / exec 都汇入 agent.Run
  • • 每轮：分区工具 → 压缩 → 流式 → 执行工具
  • • 工具受 Sandbox + 权限门 + Hooks 约束
  • • 子系统：Provider · Tools · Sandbox · MCP · Specialist
