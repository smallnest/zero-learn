zero 交互式 TUI 流程

tui.Run → Bubble Tea 事件循环（Model/Update/View）→ 异步 agent.Run 经消息回灌 → 权限/斜杠命令交互

启动与事件循环

  • • tui.Run：TTY 检查（非终端引导去 exec）
  • • tea.NewProgram + newModel（sink=program.Send）
  • • Init → Update → View 单 goroutine 循环
  • • shift+tab 切权限 · /model /image /spec · Esc 取消

异步 agent 运行

  • • 回车 → submitPrompt → beginRun → runAgent
  • • agent.Run 在 tea.Cmd 协程后台跑
  • • 默认 PermissionModeAsk，写/bash 经权限门
  • • RequireCompletionSignal 关闭（区别于 exec）

回调回灌与交互

  • • 回调 → runtimeMessageSink → program.Send
  • • 事件以 tea.Msg 回到 Update 刷新 transcript
  • • 权限请求阻塞在 decision channel 等 UI 决定
  • • 会话事件落 SessionStore，支持 /resume
